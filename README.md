# Zombicide Scenario Editor - Landing Page

Cette landing page présente l'éditeur de scénarios Zombicide, un outil permettant aux joueurs de créer et partager leurs propres scénarios pour le jeu de société Zombicide.

## Fonctionnalités Présentées

- Import de jeux via fichiers ZIP
- Éditeur de cartes intuitif
- Placement de tuiles, portes et objectifs
- Système de sauvegarde des scénarios
- Partage communautaire

## Structure du Projet


## Installation

1. Clonez ce repository
2. Ouvrez `index.html` dans votre navigateur

## Développement

Cette landing page utilise :
- HTML5 sémantique
- CSS3 moderne avec variables et flexbox/grid
- JavaScript vanilla pour les interactions
- Design responsive pour mobile et desktop
