### Description
Permettre aux utilisateurs de basculer entre deux modes :
- Mode éditeur de cartes (actuel)
- Mode éditeur de scénario (à créer)

### Acceptation criteria
- [ ] Un bouton ou menu permet de passer du mode cartes au mode scénario
- [ ] Un bouton ou menu permet de revenir du mode scénario au mode cartes
- [ ] L'état de chaque éditeur est préservé lors du basculement
- [ ] L'interface indique clairement le mode actif

### Priorité
🔴 Haute - C'est la première fonctionnalité à déployer
